# **App Name**: Adish: Mood & Time Aware Content

## Core Features:

- Mood Input System: Simple, swipe-based UI for mood selection with default moods: Focus, Relax, Bored, Happy.
- Time Intelligence Module: Automatically detect the current system time and categorize it into Morning, Afternoon, Evening, and Night.
- Hybrid Content Recommendation: A rule-based system for new users, transitioning to ML-based recommendations adapting to mood, time, and past behavior.
- Daily User Data Collection: Log every interaction including time slot, mood selected, content type shown, user action, and session duration for analysis.
- Auto Mood Suggestion: After a learning phase, the system predicts the probable user mood and displays suggestions, with options to accept or override.
- Content Display System: Show recommended content as clean cards, including categories like Study, Motivation, Music, Meditation, and Entertainment. Can use static or API-based content.
- Insights & Analytics Dashboard: Weekly summary of most selected moods, consumed content, and peak productivity times, displayed with simple charts. Use tool to incorporate additional content that may be useful for insights

## Style Guidelines:

- Primary color: Slate blue (#7395AE) evokes a sense of calm and focus.
- Background color: Very light blue (#E8EBEF), provides a soft backdrop that does not distract.
- Accent color: Light gold (#D0B49F), offering subtle contrast and a touch of warmth.
- Body and headline font: 'PT Sans', a modern, humanist sans-serif suitable for headlines and body text.
- Minimalist icons that represent moods and content categories.
- Smooth, subtle animations for mood transitions and content loading.
- Clean and distraction-free layout, with a clear "Suggested for You" section.
